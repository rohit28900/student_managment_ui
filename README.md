# Student Management UI

Modern Student Management System built using:

- Next.js
- Tailwind CSS
- shadcn/ui
- Playwright
- FastAPI Integration

## Features

- Authentication
- Dashboard
- Student Management
- Courses
- Attendance
- Profile & Settings
- Playwright UI Testing

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

Testing:
- Playwright

Backend:
- FastAPI
